# ESL 學習單生成器（4 歲）

給 4 歲幼兒英語學習者的入門練習單，**不太需要認字能力**，共 9 種：

1. **圖片配對（看圖找一樣）** — 純圖案配對，完全不需要認字
2. **數字與單字練習** — 數一數圖案、描寫數字，認識單字
3. **著色學習單** — 顏色單字＋對應形狀，孩子照單字塗上顏色
4. **單字描寫練習單** — 單字＋emoji 圖示，四線描寫格（建議用短單字、少描寫次數）
5. **分類練習單** — 兩個類別打散成單字庫，有 emoji 輔助不完全依賴認字
6. **找不同** — 一排圖案中圈出不一樣的那一個，純觀察力不需認字
7. **完整 ABC 字母描寫** — A-Z 全部 26 個字母大小寫描寫，不需輸入單字
8. **情緒單字配對** — 表情 emoji 配對情緒單字（happy/sad/angry...）
9. **反義詞配對** — 大/小、熱/冷等相反詞的圖片配對

只有兩個檔案：`index.html`、`modules.js`，不需要任何後端或資料庫。

> 這是「學習單」系列中專給 **4 歲** 設計的版本。5-6 歲需要認字能力的類型（字母配對、Sight Words、字首音練習等），請看另一個 repo：**ESL 學習單生成器（5-6 歲）**。教具／課堂互動遊戲請看 **ESL 教具生成器**。

## 上架 GitHub 完整步驟

1. 登入 [github.com](https://github.com)
2. 右上角 `+` → `New repository`，取個名字（例如 `esl-worksheets-age4`），設為 **Public**
3. 點 `Add file` → `Upload files`，把 `index.html` 和 `modules.js` 一起拖進去 → `Commit changes`
4. 進 `Settings` → `Pages` → Branch 選 `main`，資料夾選 `/ (root)` → `Save`
5. 等 1-2 分鐘，會拿到網址：`https://你的帳號.github.io/esl-worksheets-age4/`

## 使用小提醒

- **列印**：每個學習單右上角有「列印」按鈕，只印練習單本體，版面已抓好 A4 邊界
- **圖片配對**：格式 `單字:emoji`，單字只是給老師參考，孩子只看圖案
- **數字與單字**：格式 `單字:emoji:數量`，例如 `apple:🍎:3`，數量上限 10
- **單字描寫**：建議單字長度 3-5 字母，描寫次數設 2-3 次即可，太多對 4 歲來說會失去耐心
- **找不同**：格式 `多數:少數`，例如 `cat:dog`，可調整每排幾個圖案
- **ABC 全字母描寫**：不用輸入任何內容，選範圍（A-M / N-Z / 全部）直接產生
- **情緒配對 / 反義詞配對**：情緒配對格式 `單字:emoji`；反義詞配對格式是 `單字1:emoji1:單字2:emoji2`
